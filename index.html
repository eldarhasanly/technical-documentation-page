Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of web pages. It allows developers to separate content from design, making web pages more visually appealing and easier to maintain. CSS can be applied directly to HTML elements, included within HTML documents, or linked externally.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • A solid understanding of HyperText Markup Language (HTML), including its structure and semantics.
  • Some familiarity with web development concepts.
CSS and HTML

CSS works alongside HTML to define the appearance of web elements. While HTML provides the structure of a webpage, CSS dictates how those elements should be styled, including colors, fonts, spacing, and positioning.

Hello World

To get started with CSS, you can create a simple stylesheet and apply it to an HTML document.

Example:

            
              !DOCTYPE html
              html lang="en"
              head
                meta charset="UTF-8"
                meta name="viewport" content="width=device-width, initial-scale=1.0"
                title Hello World CSS /title
                style
                  body {
                    background-color: lightblue;
                    font-family: Arial, sans-serif;
                  }
                /style
              /head
              body
                h1 Hello, World! /h1
              /body
              /html
Selectors

CSS selectors define which elements to style. Some common types include:

  • Element Selector: Targets all elements of a specific type (e.g., p { color: blue; }).
  • Class Selector (.): Targets elements with a specified class (e.g., .box { border: 1px solid black; }).
  • ID Selector (#): Targets an element with a unique ID (e.g., #header { font-size: 24px; }).
  • Attribute Selector: Targets elements based on attributes (e.g., [type="text"] { background-color: yellow; }).
Box Model

CSS uses the box model to represent elements. It consists of:

  • Content: The actual text or image within the element.
  • Padding: Space around the content.
  • Border: A surrounding border.
  • Margin: Space between the element and others.

Example:

            
              div {
                width: 200px;
                padding: 20px;
                border: 5px solid black;
                margin: 10px;
              }
Positioning

CSS provides various ways to position elements:

  • Static (default): Elements appear in the normal document flow.
  • Relative: Positioned relative to their normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport and does not move when scrolling.
  • Sticky: Switches between relative and fixed depending on scroll position.

Example:

            
              .fixed-box {
                position: fixed;
                top: 0;
                right: 0;
                background-color: red;
              }
Flexbox and Grid

Modern CSS layout techniques include:

  • Flexbox: A one-dimensional layout system for aligning items along a row or column.
  • Grid: A two-dimensional layout system for creating complex layouts.

Example (Flexbox):

            
              .container {
                display: flex;
                justify-content: center;
                align-items: center;
              }

Example (Grid):

            
              .grid-container {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
              }
Colors and Fonts

CSS allows you to customize colors and fonts using properties like color, background-color, font-family, and font-size.

Example:

            
              body {
                color: #333;
                background-color: #f5f5f5;
                font-family: 'Arial', sans-serif;
              }
Responsive Design

CSS makes it easy to create responsive designs using media queries.

Example:

            
              @media (max-width: 600px) {
                body {
                  background-color: lightgray;
                }
              }
Animations

CSS supports animations and transitions for smooth effects.

Example:

            
              .box {
                width: 100px;
                height: 100px;
                background-color: blue;
                transition: background-color 0.5s ease-in-out;
              }

              .box:hover {
                background-color: red;
              }
Conclusion

CSS is a powerful tool for styling web pages. By understanding selectors, layout techniques, and responsive design, you can create visually appealing websites that work across different devices.
